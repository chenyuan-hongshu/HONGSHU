# 可视化界面运行说明

## 1. 系统概述

基于 Flask 的糖尿病风险预测交互系统，集成 CatBoost 预测模型 + Qwen2.5 大语言模型，支持风险评估和AI健康对话。

## 2. 技术架构

```
浏览器（前端HTML/CSS/JS）
    ↕ HTTP请求
Flask后端（Python）
    ├── CatBoost模型 → 风险预测
    ├── SHAP解释器 → 特征贡献分析
    └── Qwen2.5-1.5B → AI健康对话
```

## 3. 安装依赖

```bash
pip install flask pandas numpy scikit-learn catboost shap matplotlib
pip install torch transformers accelerate bitsandbytes
```

## 4. 模型文件准备

### 4.1 CatBoost模型（必需）
运行以下脚本生成：
```bash
python 数据加载与预处理.py    # 生成清洗数据
python 模型训练.py            # 生成基线模型
python 模型调优.py            # 生成最终优化模型
```

生成的模型文件位于 `output/models/catboost_final.pkl`

### 4.2 Qwen大模型（AI对话功能）
首次启动时会**自动检测并下载**Qwen模型（约3.6GB），下载来源为HuggingFace国际开源平台，模型开源可审计，安全无风险。

如需手动下载或网络较慢，可设置国内镜像：
```bash
set HF_ENDPOINT=https://hf-mirror.com
```

> 注：Qwen模型为可选项。下载失败时风险预测功能正常，AI对话显示提示信息。

## 5. 启动方式

```bash
cd 可视化
python app.py
```

首次启动会自动下载Qwen模型（约3.6GB），终端会显示明确的下载进度。下载完成后浏览器打开 `http://localhost:5000`

## 6. 页面功能

### 左侧导航栏
| 模块 | 说明 |
|------|------|
| 风险预测 | 核心功能：填写指标 → 风险评估 → AI健康对话 |
| 描述性分析 | 12张EDA图表 + 文字分析（数据概况、人口学、健康指标、生活习惯、交叉分析、相关性） |
| 模型对比 | 7种模型性能对比、ROC曲线、PR曲线、混淆矩阵 |
| 模型调优 | 不平衡方案对比、CatBoost调参、SHAP可解释性分析、Top5风险因子 |

### 风险预测页面（聊天式交互）
1. 上方填写健康指标（14个二分类 + 7个数值特征）
2. 点击「计算风险」→ 右侧显示用户消息，左侧弹出评估结果
3. 在输入框输入健康问题 → 基于评估结果给出个性化建议

### AI对话原理
- 用户提问时，系统将预测结果（风险等级、概率、高血压、BMI、吸烟等）注入提示词
- Qwen模型基于完整用户档案生成针对性健康建议
- 对话历史自动保留最近2轮，避免越聊越慢

## 8. 常见问题

### Qwen加载报错"页面文件太小"
增大Windows虚拟内存：
1. 右键「此电脑」→ 属性 → 高级系统设置
2. 性能 → 设置 → 高级 → 虚拟内存 → 更改
3. 自定义大小：初始8192MB，最大16384MB
4. 重启电脑

### AI回复速度慢
- 正常速度：10-15秒/回复（1.5B模型在笔记本上的合理速度）
- 确保GPU可用：启动时终端显示 `运行设备: cuda:0`
- 关闭其他占用显存的程序

### 预测功能报错
确保已运行前置脚本生成模型文件，检查 `output/models/catboost_final.pkl` 是否存在。
