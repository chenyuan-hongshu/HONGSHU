# 探索性数据分析（EDA）说明文档

## 1. 分析目标

基于清洗后的 `diabetes_cleaned.csv` 数据集，通过描述性统计、单维度分析、多维度交叉分析和全局相关性分析，全面了解糖尿病患病的影响因素，为后续建模提供业务支撑。

## 2. 数据概况

| 项目 | 数值 |
|------|------|
| 总样本量 | 253,680 条 |
| 特征数量 | 21 个 |
| 基线患病率 | 13.94% |

数据来源：CDC BRFSS 2015 糖尿病健康指标数据集。

## 3. 分析流程与产出

### 3.1 整体描述性统计

**做了什么：**
- 计算全数据集各字段的均值、中位数、标准差、最小值、最大值及分位数
- 统计正负样本（糖尿病/未患病）的数量与占比

**得到了什么：**
- `descriptive_stats.csv` — 可直接复用进报告的统计结果表
- 基线患病率：13.94%，属于典型的类别不平衡数据集

### 3.2 单维度特征与患病关联分析

**做了什么：** 逐个分析核心特征与糖尿病患病率的关系，每个特征对应一张可视化图表。

| 分析维度 | 分析内容 | 产出图表 |
|----------|----------|----------|
| 人口学特征 | 各年龄组患病率对比 | `01_年龄分层患病率.png` |
| 人口学特征 | 男性/女性患病率对比 | `02_性别患病率对比.png` |
| 健康指标 | BMI分布（患病/未患病） | `03_BMI分布直方图.png` |
| 健康指标 | 高血压、高胆固醇患病率对比 | `04_高血压高胆固醇患病率.png` |
| 生活习惯 | 吸烟/不吸烟患病率对比 | `05_吸烟患病率对比.png` |
| 生活习惯 | 有/无体育活动患病率对比 | `06_体育活动患病率.png` |
| 生活习惯 | 蔬果摄入患病率对比 | `07_蔬果摄入患病率.png` |

**主要发现：**
- 患病率随年龄增长显著上升，60岁以上人群风险明显增加
- 男性患病率略高于女性
- 糖尿病患者BMI分布整体偏高，肥胖人群患病风险显著增加
- 有高血压人群患病率远高于无高血压人群
- 吸烟人群患病率高于不吸烟人群
- 不运动人群患病率显著高于运动人群

### 3.3 多维度交叉分析

**做了什么：** 构建双因素交叉分组，分析特征组合的叠加效应。

| 交叉组合 | 分析目的 | 产出图表 |
|----------|----------|----------|
| BMI肥胖 + 高血压 | 分析两者叠加对患病率的影响 | `08_BMI高血压交叉分析.png` |
| 高龄 + 吸烟 | 分析高龄吸烟者的患病风险 | `09_高龄吸烟交叉分析.png` |
| 运动习惯 + 肥胖 | 分析运动是否能抵消肥胖风险 | `10_运动肥胖交叉分析.png` |

**主要发现：**
- BMI肥胖与高血压叠加后患病率最高，呈现显著协同效应
- 高龄吸烟者的患病风险远高于其他组合
- 即使肥胖，保持运动也能在一定程度上降低患病风险

### 3.4 全局相关性分析

**做了什么：**
- 计算所有特征与目标变量的皮尔逊相关系数
- 绘制全特征相关性热力图
- 按关联强度排序输出Top10核心影响特征

**产出图表：**
- `11_相关性热力图.png` — 全特征相关性矩阵
- `12_Top10相关特征.png` — 按相关系数绝对值排序的Top10特征

**核心影响特征Top5：**

| 排名 | 特征 | 相关系数（绝对值） |
|------|------|-------------------|
| 1 | 高血压 | 0.2635 |
| 2 | 自评健康 | 0.2583 |
| 3 | BMI | 0.2183 |
| 4 | 年龄 | 0.2116 |
| 5 | 心脏病 | 0.2093 |

## 5. 对后续建模的指导意义

1. **特征选择**：高血压、自评健康、BMI、年龄、心脏病等特征与糖尿病相关性最强，应重点关注
2. **类别不平衡**：正负样本比例约1:6，建模时需采用SMOTE过采样、代价敏感学习等方法处理
3. **非线性关系**：BMI、年龄等连续特征与患病率存在非线性关系，树模型可能优于线性模型
4. **交互效应**：BMI+高血压、高龄+吸烟等组合具有协同效应，可考虑构造交互特征
