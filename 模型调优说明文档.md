# 模型调优说明文档

## 1. 本阶段目标

围绕最优的 CatBoost 模型，完成类别不平衡处理、参数调优、评估体系补全、可解释性分析四个核心步骤，将建模部分从基础对比升级到有深度、有优化、有业务结论的优秀水平。

## 2. 实验流程概览

| 步骤 | 内容 | 核心目的 |
|------|------|----------|
| 步骤1 | 类别不平衡方案对比 | 解决正负样本失衡问题，提升召回率 |
| 步骤2 | CatBoost专项调参 | 挖掘模型性能上限 |
| 步骤3 | PR曲线评估体系 | 补全ROC之外的评估维度 |
| 步骤4 | SHAP可解释性分析 | 打开黑盒，输出业务结论 |

## 3. 步骤1：类别不平衡方案对比

### 3.1 实验设计

控制变量：固定 CatBoost 模型结构和测试集不变，仅改变训练集处理方式。

| 方案 | 处理方式 | 是否改动数据 | 优缺点 |
|------|----------|-------------|--------|
| 基准（原始） | 直接使用原始训练集 | 否 | 基准参照 |
| 代价敏感学习 | `auto_class_weights='Balanced'` | 否 | 无信息损失，不改变数据分布 |
| SMOTE过采样 | 合成少数类样本至1:1 | 是 | 人工合成新样本，可能引入噪声 |

### 3.2 评估指标

统一使用召回率、AUC、F1值三个核心指标，优先看召回率提升。

### 3.3 输出文件

| 文件 | 说明 |
|------|------|
| `类别不平衡方案对比.csv` | 三种方案的指标对比表 |
| `catboost_imbalance_best.pkl` | 最优不平衡方案对应的模型 |

## 4. 步骤2：CatBoost专项调参

### 4.1 调参策略

- **调参方式**：随机搜索（RandomizedSearchCV），20轮迭代
- **交叉验证**：3折分层交叉验证
- **优化目标**：召回率（医疗筛查不漏诊优先）

### 4.2 调参空间

| 参数 | 搜索范围 | 说明 |
|------|----------|------|
| `learning_rate` | [0.01, 0.03, 0.05, 0.1, 0.15] | 学习率 |
| `depth` | [4, 6, 8, 10] | 树深度 |
| `iterations` | [200, 300, 500] | 迭代轮数 |
| `l2_leaf_reg` | [1, 3, 5, 7, 9] | L2正则化系数 |
| `bagging_temperature` | [0, 0.5, 1, 2] | 样本采样比例 |

### 4.3 输出文件

| 文件 | 说明 |
|------|------|
| `调参前后对比.csv` | 调参前后的召回率/AUC/F1对比 |
| `catboost_final.pkl` | 最终优化后的CatBoost模型 |
| `best_params.pkl` | 最优参数组合 |

## 5. 步骤3：PR曲线评估体系

### 5.1 为什么需要PR曲线

- ROC曲线在类别不平衡场景下会高估模型性能
- PR曲线（精确率-召回率曲线）更能真实反映模型在少数类上的识别能力
- AP值（PR曲线下面积）是评估不平衡数据集的核心指标

### 5.2 对比模型

6个模型统一绘制PR曲线并计算AP值：

| 模型 | 说明 |
|------|------|
| 逻辑回归 | 线性模型基准 |
| 随机森林 | Bagging集成代表 |
| XGBoost | GBDT代表 |
| LightGBM | GBDT代表 |
| CatBoost基准 | 调参前的CatBoost |
| 最终最优模型 | 调参后的CatBoost |

### 5.3 输出文件

| 文件 | 说明 |
|------|------|
| `PR曲线对比.png` | 6个模型的PR曲线叠加对比图 |
| `PR曲线AP值对比.csv` | 各模型AP值排序表 |

## 6. 步骤4：SHAP可解释性分析

### 6.1 分析设置

- 使用最终最优模型（调参后CatBoost）
- 从测试集中随机抽取1000条样本
- 使用TreeExplainer计算SHAP值

### 6.2 三类分析结果

| 分析类型 | 说明 | 产出图表 |
|----------|------|----------|
| 全局特征重要性 | 按平均\|SHAP值\|排序的Top20特征 | `SHAP_全局特征重要性.png` |
| 特征影响摘要 | 每个特征取值高低对风险的影响方向和强度 | `SHAP_特征影响摘要.png` |
| 单样本个体解释 | 高风险样本的逐特征贡献拆解 | `SHAP_单样本解释.png` |

### 6.3 SHAP值解读

- **正SHAP值**：推动模型预测为"患病"
- **负SHAP值**：推动模型预测为"未患病"
- **红色**：特征取值高（如BMI高、年龄大）
- **蓝色**：特征取值低（如BMI低、年龄小）

### 6.4 输出文件

| 文件 | 说明 |
|------|------|
| `SHAP特征重要性.csv` | 特征重要性排序表 |
| `SHAP_全局特征重要性.png` | 全局特征重要性条形图 |
| `SHAP_特征影响摘要.png` | 特征影响方向摘要图 |
| `SHAP_单样本解释.png` | 单样本瀑布图 |

## 8. 业务结论

### 8.1 核心风险因子Top5

基于SHAP分析，影响糖尿病患病风险的前5个核心因子：

| 排名 | 特征 | 业务含义 |
|------|------|----------|
| 1 | 高血压 | 高血压患者患病风险显著升高 |
| 2 | 自评健康 | 自评健康越差，患病风险越高 |
| 3 | BMI | BMI越高，患病风险越大 |
| 4 | 年龄 | 年龄增长与患病风险正相关 |
| 5 | 心脏病 | 有心脏病史者风险更高 |

### 8.2 干预建议

1. **重点筛查人群**：60岁以上、有高血压/心脏病史、BMI≥30的人群
2. **生活方式干预**：鼓励体育活动、控制体重、戒烟限酒
3. **医疗资源倾斜**：对高风险人群提供定期血糖检测
